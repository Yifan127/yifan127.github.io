Access data from Restful API

Posted on 2017-08-07 | In web scraping

Data description:

Auckland Transport has a Web API for accessing real-time traffic information (real-time trip updates)
https://api.at.govt.nz/docs/v1#Realtime
You need to register to get access here …
https://api.at.govt.nz/

The goal:

  1. Call the GTFS real-time trip updates API once every minute for 60 minutes and save the JSON results.
  2. Process the JSON results to extract the trip id, route id, vehicle, departure and arrival delay, stop sequence, stop id, and timestamp for each trip update.

Functions and packages:

  1. Functions: GET(), writeLines(), content(), htmlParse(), getNodeSet(), xpathSApply(), flatten
  2. Packages: httr, jsonlite

Downloading data source

First, we use the variable i as a counter, and call the GTFS real-time trip updates API with header "Ocp-Apim-Subscription-Key". If the response code is 200, then we save the content of the response to local as a JSON file and add 1 to the counter. After that, we wait 1 minute and repeat the above steps until we get 60 JSON files.

Processing data source to a data frame

Reading all the downloaded JSON files into R.

To extract the real time trips information to a data frame, we initialize an empty list to store the data frames. In the for loop, we use the flatten function to turn the nested data frame to a normal one, and then extract and save all the information to a data frame. At the last, we convert the list to data frame.
