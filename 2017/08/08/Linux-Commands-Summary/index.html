General Linux Commands Summary

Posted on 2017-08-08 | In Linux

Directory & File

Command Description
ls list directory contents
-a all
-l long listing format
-t sort by modification time
-r reverse
cp copy files and directories
-u update, copy only newer files
scp secure copy to another host
mv move (rename) files
rm remove files or directories
-f force
-r recursive
chmod change file mode bits
- read (r), write (w), execute(x)
- the user who owns it (u), other users in the file’s group (g), other users not in the file’s group (o), or all users (a)
chown change file owner and group
- user:group

File

Command Description
cat concatenate files and print on the standard output
head output the first part of files
-n lines
tail output the last part of files
-n lines
more view the files in pages
d scroll forward
less similar to more, but allows backward movement as well as forward move
d scroll forward
b scroll backword
echo display a line of text on the screen
find search for files in a directory
find ./ -name myfile.txt
grep print lines matching a pattern
grep word /etc/passwd
wc count the lines, words, and characters in a file
-l count line
ls -l | wc -l: count the number of objects in current directory
tar -cvzf file.tar.gz /home/download: create tar.gz from files verbosely
-xvzf file.tar.gz: extract files

Process

Command Description
ps report the current processes
-a all
-f full listing
kill terminate a process
top display Linux processes
nice run a program with modified scheduling priority
-n values range from -20 to 19 (from the highest priority to the lowest) to 19

User

Command Description
who how who is logged on
whoami print effective userid
addgroup display Linux processes
adduser create a new user or update default new user information
-d home directory
-g group id
sudo execute a command as another user
-u to run the specified command as a user other than root

Network

Command Description
ping verify if the computer can connect over the network to another computer
netstat print network connections, routing tables
ip address
ifconfig		 check ip address
traceroute print the path that packets take to a network host
traceroute google.com
telnet user interface to the TELNET protocol
telnet myhost.com:5555
nslookup query Internet name servers (DNS)
nslookup google.com

Resource

Command Description
df report file system disk space usage
-h human readable
-k in KB
du estimate file space usage, summarize disk usage of each FILE, recursively for directories
free display amount of free and used memory in the system
-m in MB

Install package

Command OS Description
yum Redhat/Centos install, remove, update, info, list, search
apt-get Debian/Ubuntu update： resynchronizes the local index of packages files
install, remove, upgrade

Environmental Variables

Command Description
printenv print all or part of environment
export set env variable
export PATH=${PATH}:/home/xxx
vi ~/.bash_proflle
append the PATH		 make all settings permanent

VIM Text Edit

Command Description
i insert mode
a append mode
0(zero) move cursor to start of current line
$ move cursor to end of current line
yy copy line in buffer
p paste/put buffer
x delete character
dd delete line
:q! quit no save
:wq quit and save

Help

Command Description
man see the usage of a command
# shell
Access Data From Restful API
Run Unix Shell Command in R
