Access data from web pages

Posted on 2017-08-06 | In web scraping

Data description:

The “HOT 100” billboard web site contains the top 100 songs for each week. The URL for the week starting Sat 29 July (2017-07-29) is http://www.billboard.com/charts/hot-100/2017-07-29.

The goal:

  1. Download the HOT 100 for the weeks so far in 2017 (from Sat 7 Jan to Sat 29 July).
  2. Use XPath to extract the rank, song, and artist, for the top 100 songs for each week.

Functions and packages:

  1. Functions: seq(), trimws(), order(), htmlParse(), getNodeSet(), xpathSApply()
  2. Packages: XML

Downloading data source

First, we get the date of all the Saturdays from 2017-01-07 to 2017-07-29, then use sprintf to get the urls.

To download the data, we use htmlParse in the XML package. It parses the html file to an object ‘XMLInternalDocument’. After that, we use getNodeSet to find all the article nodes by its Xpath. The Xpath ‘//article[contains(@class, ‘chart-row’)]’ matches all the

elements with a class attribute that contains chart-row.

Processing the data to a data frame

The following function is used to convert xml to data frame. There is only one argument in this function called articleNode. In this function, we extract rank, song and artists within an article node by their Xpath. Notice that the artist element is contained within either an <a> or a <h3> element. Also, we use trimws to remove all the whitespace in artist, including [ \t\r\n]. The output of this function is a data frame which contains the rank, song and artist.

We apply the above function to all the article nodes, and get a list which contains 30 elements, and each element is a data frame. Finally, we convert the list to a data frame, and add the date column to the data frame.

We can now calculate the average rank for each artist, and order them by the average rank.

After that, we wait 1 minute and repeat the above steps until we get 60 JSON files.
